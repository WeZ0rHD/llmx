# LLMX demo project

This is a tiny, **already-initialized** LLMX project used for tests and
demos. Run from this directory:

```bash
llmx status
llmx task list
llmx decision list
llmx session show
```

To re-render the entry-point files for the target agents:

```bash
llmx export claude
llmx export codex
```

The directory layout is the canonical v0.1 format — see
`../../specs/llmx-v0.1.md` for details.
